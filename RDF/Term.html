Overview

An RDF term.

Terms can be used as subjects, predicates, objects, and graph names of statements.

def term?(*args)
  case args.length
  when 0 then true
  when 1 then false
  else raise ArgumentError("wrong number of arguments (given #{args.length}, expected 0 or 1)")
  end
end
